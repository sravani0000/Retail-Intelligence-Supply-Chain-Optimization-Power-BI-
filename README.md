Project Overview: Built a multi-regional executive dashboard for the Canada Superstore dataset to provide visibility into sales KPIs and regional demand fluctuations.

Technical Highlights:

Advanced Visualization: Developed Geospatial Filled Maps to track revenue density across provinces and Area Charts to monitor quantity trends over a multi-year period.

DAX Engineering: Implemented custom DAX measures for Year-over-Year (YoY) growth and moving averages, enabling stakeholders to perform real-time trend forecasting.

Storytelling: Integrated Q&A features and drill-through actions to allow executives to self-serve deep-dive analysis on specific product categories.
